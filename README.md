# Last Overhead Prayer

A RuneLite plugin that displays the last overhead prayer you activated, along with whether it is currently active or not.

## What it does

Shows a small overlay displaying:
- The name of the last overhead prayer you used (e.g. "Protect Magic")
- Whether that prayer is currently **ON** (green) or **OFF** (red)

This is useful during PvM encounters where you need to prayer flick or swap between overhead prayers and want a quick visual reminder of what you last had active.

## Supported prayers

- Protect from Melee
- Protect from Missiles
- Protect from Magic
- Retribution
- Smite
- Redemption

## Configuration

| Option | Description | Default |
|--------|-------------|---------|
| Hide when active | Hides the overlay while the prayer is currently on — only shows when it's off | Off |
| Show prayer name | Toggles the prayer name text | On |
| Active color | Color of the text when the prayer is currently active | Green |
| Inactive color | Color of the text when the prayer is not active | Red |

## Screenshot

The overlay appears in the top-left corner of the screen by default and can be repositioned by holding Alt and dragging it.
